# app/ — 미리봄 V1.1 작업 입구

이 폴더는 **V1.1(Supabase 연동 버전)을 만드는 작업 공간**입니다.

- **원본 `../index.html`** = V1.0. 백엔드 없는 뷰어 단독 버전. **건드리지 않고 보존**합니다.
- **이 폴더 `app/index.html`** = 위 원본을 복사한 시작점. 앞으로 여기에
  로그인·저장·공유·코멘트(Supabase 연동) 기능을 붙여 나갑니다.

## 주소(입구) 구분

| 입구 | 버전 | 비고 |
|---|---|---|
| `/miribom/` (루트 index.html) | V1.0 | 공개 데모. 그대로 유지 |
| `/miribom/app/` (이 폴더) | V1.1 | 작업 중. 기능 연동 예정 |

## 현재 상태

지금은 원본을 그대로 복사만 한 상태라 두 입구의 화면이 동일합니다.
2단계(화면 ↔ 백엔드 연결)에서 이 `app/index.html` 만 수정합니다.

> 참고: 이 V1.1 작업은 `v1.1` 브랜치에서 진행됩니다. 공개 웹사이트(GitHub Pages)는
> `main` 브랜치를 보여주므로, 여기 변경은 나중에 `v1.1 → main` 으로 합칠 때 공개됩니다.

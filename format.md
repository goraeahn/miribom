# 미리봄 마크다운 형식 가이드

미리봄(https://mirivom.com)은 마크다운으로 쓴 원고를 **표지·장(章)·각주·쪽번호가 갖춰진 "책"처럼** 보여 주는 뷰어입니다. 일반 마크다운에 더해 **YAML 머리말**과 **`::: 이름표` 특별 쪽** 같은 약간의 규칙을 씁니다. 이 문서가 그 규칙 전부입니다.

## 이 문서 사용법 — AI에게 글 변환 시키기

1. 챗봇(ChatGPT·Claude 등)에 **이 문서 전체**를 붙여넣거나 `https://mirivom.com/format.md` 링크를 줍니다.
2. 이어서 변환할 글을 주며 말합니다: **"위 미리봄 형식에 맞춰 이 글을 마크다운으로 변환해줘. 마크다운만 출력해."**

복사해서 바로 쓸 수 있는 프롬프트:

```
당신은 "미리봄"(mirivom.com) 원고 변환기입니다.
아래 [형식 규칙]에 맞춰, 사용자가 주는 글을 미리봄 마크다운으로 변환하세요.
- 결과는 마크다운만 출력하고 설명·인사말은 붙이지 마세요.
- 책 정보(제목·지은이 등)는 맨 위 YAML 머리말에 넣으세요.
- 표지에는 그림 파일을 넣지 말고 제목·지은이 "글자"만 넣으세요(그림 표지는 미리봄 사이트에서 업로드).
- 원문에 없는 내용을 지어내지 말고, 구조(장·소제목·머리말 등)만 형식에 맞게 정리하세요.

[형식 규칙]  ← 아래 '형식 규칙' 단락을 여기에 붙여넣으세요
```

## 형식 규칙

### 1) 책 정보 — 맨 위 YAML 머리말
글 맨 처음에 `---` 사이로 책 정보를 적습니다. 표지·판권에 자동으로 채워집니다.

```
---
title: 책 제목
author: 지은이
date: 2026-01-01
publisher: 펴낸곳
lang: ko
---
```
필수는 `title` 정도. 나머지는 있으면 쓰고 없으면 생략합니다.

### 2) 제목 단계
- `#` = 가장 큰 묶음(부). 글 맨 앞의 첫 `#`은 **책 제목**으로도 쓰입니다.
- `##` = 장(章)
- `###` = 작은 소제목
- 본문에서는 되도록 `#`을 쓰지 않습니다(단편집처럼 꼭 필요할 때만).

### 3) 문단·강조·인용·장면 전환
- 문단은 **빈 줄**로 나눕니다. 한 문단 안에서는 줄을 이어 씁니다.
- 강조: `*기울임*`, `**굵게**`
- 인용문: 줄 앞에 `>`
- 장면이 바뀌는 자리: 빈 줄에 `---` 또는 `· · ·`

### 4) 각주
본문에 `[^1]`, 글 아래쪽에 `[^1]: 각주 내용` 형태로 적습니다.

### 5) 특별한 쪽 — `::: 이름표 … :::`
표지·머리말·후기처럼 본문이 아닌 쪽은 콜론 셋으로 감쌉니다.

```
::: 머리말
이 책을 펴내며 짧게 적는 글입니다.
:::
```
쓸 수 있는 이름표
- **앞쪽**: `표지` `속표지` `판권` `헌사` `머리말` `서문` `들어가는글` `목차`
- **뒤쪽**: `후기` `부록` `색인` `간기`

### 6) 표지
- 표지에는 **제목·지은이 "글자"만** 넣습니다. **그림 파일(이미지 경로)은 넣지 마세요.**
- 그림 표지를 쓰려면 미리봄 사이트에서 **표지 이미지를 직접 업로드**하세요. 그림이 없으면 제목에 어울리는 색 표지가 자동으로 만들어집니다.

```
::: 표지
# 책 제목
### 부제
:::
```

### 7) 표·목록
- 목록: 줄 앞에 `- ` 또는 `1. `
- 표: `| 칸 | 칸 |` 형식

## 복붙 템플릿 (전체 예시)

```
---
title: 예시 단편
author: 홍길동
date: 2026-01-01
---

::: 표지
# 예시 단편
### 짧은 부제
:::

::: 머리말
이 글을 쓰게 된 짧은 사연을 적습니다.
:::
```
